# IG Format QA Checklist V3

## Format Integrity
- [ ] Carousel assets are true multi-image sets (not faux slides).
- [ ] Reel assets include on-camera script and subtitle file.
- [ ] No single static post is pretending to be a carousel.

## Cover Performance
- [ ] Cover hook creates tension/curiosity.
- [ ] Cover text is readable in 1 second.
- [ ] Cover contrast holds at thumbnail size.

## Slide Readability
- [ ] One clear sentence per slide.
- [ ] No slide exceeds ~12 words.
- [ ] No paragraph blocks.

## Design System
- [ ] Layered background treatment used (no boxed template look).
- [ ] Brazil imagery applied with restraint and readability.
- [ ] Max two fonts used across all assets.

## Watermark Controls
- [ ] Watermark size ~10–12% width.
- [ ] Bottom-right placement consistent.
- [ ] 32px margin from right and bottom.

## Tone + Positioning
- [ ] Adult learner voice.
- [ ] No teacher/course framing.
- [ ] Calm, practical, non-hype copy.

## Hashtags + CTA
- [ ] 3–5 hashtags per post.
- [ ] Includes branded tag `#NoahFalaPortugues`.
- [ ] CTA varies by content type.

## Approval + Audit
- [ ] Asset filenames match upload mapping exactly.
- [ ] Pre-upload screenshot captured.
- [ ] Post-upload screenshot captured.
- [ ] Summary file correctly states whether any upload occurred.
