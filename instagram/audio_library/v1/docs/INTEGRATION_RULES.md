# Music Division Integration Rules

## Default
1. Reels default to NFP internal music library.
2. IG native music is optional experiment only.

## Audio Priorities
1. Voice clarity is first priority.
2. Music should support tone, not dominate attention.
3. No vocals in backing tracks.

## Operating Flow
1. CS selects track from TRACK_REGISTRY.csv.
2. Noah records reel.
3. Audio is applied in edit or inside IG as needed.
4. QA checks voice/music balance before posting.

## Success Markers
- consistent sound across posts
- recognizable tone across reels
- no copyright/licensing conflicts
