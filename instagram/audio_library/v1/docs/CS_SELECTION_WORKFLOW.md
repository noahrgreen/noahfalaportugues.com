# CS Workflow — Music Selection for Reels

## Default Rule
Use internal NFP track library as default for all reels.
IG native music remains optional experiment only.

## Selection Steps
1. CS picks one track from TRACK_REGISTRY.csv based on reel intent.
2. Noah records on-camera reel.
3. In edit, place selected track under voice.
4. Set background music level so voice remains primary.
5. Export draft and run clarity QA before upload.

## Quick Match Guide
- Calm reflection / learning update: NFP_TRK_001 or NFP_TRK_003
- Study process / SRS topics: NFP_TRK_002 or NFP_TRK_005
- BJJ phrase / mat context: NFP_TRK_004

## Audio Mix Guardrails
- Voice must remain dominant at all times.
- Recommended background target: about -24 dB under voice.
- If consonants feel masked, reduce music by additional 2-4 dB.
