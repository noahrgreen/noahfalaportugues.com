# IG Format QA Checklist V3

## Cover Strength
- [ ] Cover hook creates curiosity/tension.
- [ ] Cover text contrast is strong at first glance.

## Slide Readability
- [ ] One clear sentence per slide.
- [ ] No dense paragraphs.
- [ ] Scannable in under 2 seconds.

## Format Correctness
- [ ] Carousel content shipped as true multi-slide set.
- [ ] Reel content includes on-camera script + subtitle file.
- [ ] No faux-slide single static post.

## Brand + Tone
- [ ] Adult learner voice.
- [ ] No teacher/course framing.
- [ ] Calm, direct delivery.

## Watermark + Design
- [ ] Watermark size ~10–12% width.
- [ ] Bottom-right at 32px/32px.
- [ ] Visual system consistent with site palette.

## Hashtags + CTA
- [ ] 3–5 hashtags only.
- [ ] Includes branded tag `#NoahFalaPortugues`.
- [ ] CTA rotated by format.

## Audit Controls
- [ ] Asset filename exactly matches upload mapping.
- [ ] Pre-upload screenshot captured.
- [ ] Post-upload screenshot captured.
- [ ] Summary accurately states if upload occurred.
