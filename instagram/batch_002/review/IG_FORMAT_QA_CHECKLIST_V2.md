# Instagram Format QA Checklist (V2)

## Format Integrity
- [ ] If labeled slide content, asset exists as true carousel set (4+ files).
- [ ] No single-image post contains faux slide sequence.
- [ ] Reel items include storyboard/draft and are not static substitute posts.

## Language + Content
- [ ] Portuguese accents verified (ex: posição).
- [ ] EN/PT translations are natural and context-correct.
- [ ] Tone is adult learner, not teacher/course persona.

## Design
- [ ] 1080x1080 output for all feed assets.
- [ ] Text remains within safe margin (80px).
- [ ] Readability is clear at thumbnail scale.
- [ ] Brazil imagery adds atmosphere without clutter.

## Watermark
- [ ] Uses centralized source `instagram/assets/NFP_logo.png`.
- [ ] Bottom-right placement, 32px / 32px padding.
- [ ] Consistent size across all posts in batch.
- [ ] Aspect ratio preserved; no distortion.

## Approval + Auditability
- [ ] Review packet labels each item as carousel/reel/story.
- [ ] Asset-to-upload mapping exists and is complete.
- [ ] Upload is blocked unless approved filename exactly matches mapped filename.
- [ ] Pre-upload screenshot captured.
- [ ] Post-upload screenshot captured.
- [ ] Summary states whether upload occurred.
