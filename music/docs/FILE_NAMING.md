# File Naming System

Category file format:
- `brazil_ambient_01.mp3`
- `study_lofi_01.mp3`
- `cinematic_01.mp3`
- `bjj_01.mp3`

Master WAV format mirrors category filename in `_masters/`.

Rules:
1. Lowercase with underscores.
2. 2-digit sequence suffix.
3. Keep names stable once referenced in content mappings.
