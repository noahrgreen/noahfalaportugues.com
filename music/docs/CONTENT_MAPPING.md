# Content Matching Rules

Default mapping:
- Phrase reels -> `brazil_ambient`
- SRS / flashcard content -> `study_lofi`
- Important statement reels -> `cinematic`
- BJJ context -> `bjj`

## Core-Track Consistency Rule
At least 70% of reels should use the same 2-3 core tracks.

Recommended core set:
1. `brazil_ambient_01.mp3`
2. `study_lofi_01.mp3`
3. `bjj_01.mp3`

Use `cinematic_01.mp3` as accent track for key moments only.

## Trend Testing (Optional)
- Allow IG trending audio for roughly 1 in 5 to 1 in 7 reels.
- Trend audio must still match brand tone.
- Do not replace baseline identity tracks.
