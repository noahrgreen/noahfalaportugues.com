# Suno Prompt System (Structured)

## Global Constraints
- Instrumental only
- No vocals
- No lyrics
- 60-90 BPM
- 30-90 seconds
- Must loop cleanly
- Background-speech-safe mix

## Prompt Template
"Instrumental, no vocals, Brazilian-inspired, soft bossa nova guitar, calm and minimal, clean rhythm, no aggressive drums, suitable for background speech, warm and modern, high clarity, 60-80 BPM"
